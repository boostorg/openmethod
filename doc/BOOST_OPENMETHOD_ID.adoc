
[#BOOST_OPENMETHOD_ID]

## BOOST_OPENMETHOD_ID

### Synopsis

Defined in <boost/openmethod/macros.hpp>.

```c++
#define BOOST_OPENMETHOD_ID(NAME) /* unspecified */
```

### Description

Generates a long, obfuscated name from a short name. All the other names
generated by macros are based on this name.
