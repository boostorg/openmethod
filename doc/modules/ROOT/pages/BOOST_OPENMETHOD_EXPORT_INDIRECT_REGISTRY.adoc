
# BOOST_OPENMETHOD_EXPORT_INDIRECT_REGISTRY

Exports the state of cpp:indirect_registry[] from the module that owns it.

## Synopsis

May be defined by a program before including
`<boost/openmethod/default_registry.hpp>` (directly, or via
`<boost/openmethod.hpp>`), in _exactly one_ translation unit of the module
that owns the registry.

## Description

Same as
xref:BOOST_OPENMETHOD_EXPORT_DEFAULT_REGISTRY.adoc[BOOST_OPENMETHOD_EXPORT_DEFAULT_REGISTRY],
but for cpp:indirect_registry[], which has its own state. Client modules use
xref:BOOST_OPENMETHOD_IMPORT_INDIRECT_REGISTRY.adoc[BOOST_OPENMETHOD_IMPORT_INDIRECT_REGISTRY].

See xref:shared_libraries.adoc[Shared Libraries] for a full discussion.
